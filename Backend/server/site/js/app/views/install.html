Pomoc's 10 Minute Installation

Pomoc installation is extremely simple and will not take longer than 5 mins!




Step 1 - Download the SDK

Download the SDK from http://www.pomoc.im/sdk/download.

Step 2 - Import the SDK

Include the downloaded files PomocSDK.bundle and PomocSDK.framework into your project. You should check the Copy items into destination group's folder (if needed) option.


Click on the Xcode Project file. Go to Build Phases > Link Binary With Libraries, click on the + button and add the following frameworks and library: libicucore.dylib, Security.framework and CFNetwork.framework.


Under Build Settings, search for Other Linker Flags and add the value -ObjC.

Step 3 - Add code to your app

In your AppDelegate.m file, add the line #import <PomocSDK/PomocSDK.h> at the top.

Add the following code at the end of the - (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions method: 

- (BOOL)application:(UIApplication *)application didFinishLaunchingWithOptions:(NSDictionary *)launchOptions
{

    /* Other code here */
    [Pomoc initWithAppId:@"{{ {true: appToken, false:'<Your app ID>'}[!!appToken] }}" secretKey:@"{{ {true: appSecret, false:'<Your secret key>'}[!!appSecret] }}"];

    return YES;
}

In the View Controller that you want to allow the user to toggle the Pomoc Chat head, add #import <PomocSDK/PomocSDK.h> at the top of the file and then use the following line of code to display the Chat Head:

    [Pomoc toggleChatHead];

Step 4 - Launch!

Launch the app and try out Pomoc!
