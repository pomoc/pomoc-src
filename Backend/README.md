# Pomoc Backend

## Server
### Setup
1. `npm install`
2. `sudo npm install -g nodemon`
3. `cp settings_local.js.dist settings_local.js`
4. Modify settings_local.js for your local environment
5. `cp start.sh.dist start.sh`
6. Modify start.sh for your local environment

### Start
1. `./start.sh`
